# Building-a-Text-Difficulty-Assessment-Model_NLP2026

This repository contains the code and data used in the presentation “Building a Text Difficulty Assessment Model for Indonesian Language Education” at the 32nd Annual Meeting of the Association for Natural Language Processing (NLP2026).  

The code was created using generative AI (Gemini and ChatGPT).
